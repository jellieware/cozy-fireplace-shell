# cozy-fireplace-shell
A cozy fireplace for the shell
<br><br>
![1000970174](https://github.com/user-attachments/assets/c2466370-a6f3-4e0a-b67f-59f04483d159)

<br><br>
<br><br>
commands:
<br><br>
apt-get install ncurses<br><br>
or<br><br>
apt-get install ncurses-devel<br><br>

apt-get install build-essential<br><br>

compile:<br><br>

gcc lexsfireplace.c -lncurses -o fire<br><br>

chmod +x fire<br><br>

run with:<br><br>

bash fire<br><br>
or <br><br>
./fire
